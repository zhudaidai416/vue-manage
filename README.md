# vue-manage

## 介绍

基于 vue2 的通用后台管理系统

## 环境依赖

```bash
yarn install
```

## 运行

```bash
yarn serve
```

账号密码在文件 `api\mockServeData\permission.js` 下

## 打包

```bash
yarn build
```
